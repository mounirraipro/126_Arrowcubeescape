# Arrow Cube Escape

Arrow Cube Escape is a 3D block escape puzzle where arrow-marked cubes can only leave the board when their path is clear. The page describes the browser version plainly and avoids official-app claims, guaranteed rewards, and pressure to install software.

## Core Play Advice

Clear the outside pieces first, then use each move to open space for the blocks buried near the center.
